# DNN-PPD

This repository provides a Python implementation for **mode-profile-preserved demultiplexing based on diffractive neural networks**.

The project focuses on the design of holographic phases for fiber-mode demultiplexing while preserving the spatial profiles of optical modes. It includes modules for defining fiber optical fields, designing holographic phases, and generating output visualization results.

## Features

- Definition and generation of fiber optical fields
- Holographic phase design based on diffractive neural networks
- Encoding phase design for converting a Gaussian beam into a specified complex-amplitude optical field
- Demultiplexing phase design for mode-profile-preserved fiber-mode separation
- Output analysis and visualization

## Holographic Phase Design

The holographic phase design in this repository includes two main components:

- **Encoding phase**: encodes an input Gaussian beam into a target complex-amplitude optical field.
- **Demultiplexing phase**: separates fiber modes while preserving their output mode profiles.

## Pretrained Holographic Phases

This repository provides two `.pth` files for fiber-mode demultiplexing holographic phase design:

- 4-mode fiber-mode demultiplexing holographic phase
- 8-mode fiber-mode demultiplexing holographic phase

These pretrained files can be used to reproduce the corresponding mode-profile-preserved demultiplexing results.
